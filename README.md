# 📸 Image Search Engine

A sleek, fast, and fully responsive **Image Search Engine** powered by the **Unsplash API**. Discover millions of high-quality images with a modern UI and seamless user experience.

---

## ✨ Features

*   🔍 **Instant Search** – Find images for any keyword instantly.
*   📱 **Fully Responsive** – Optimized for Mobile, Tablet, and Desktop.
*   🎨 **Beautiful UI** – Modern glassmorphism design with CSS Grid layouts.
*   🔄 **Infinite-style Loading** – "Show More" button to load next-page results.
*   🚀 **Async Performance** – Fast, non-blocking API calls using Fetch.
*   🔝 **Back to Top** – Smooth scroll-to-top navigation.
*   ⚠️ **Error Handling** – Smart messages for empty inputs or zero results.

---

## 🛠️ Tech Stack


| Technology | Usage |

| **Unsplash API** | Real-time image data source |

## 🚀 Quick Setup

1. **Clone the project**
   ```bash
   git clone https://github.com
   ```

2. **Add your API Key**
   Open `Main.js` and replace the `AccsesKey` value with your own key from the [Unsplash Developer Portal](https://unsplash.com).

3. **Launch**
   Open `index.html` in any browser.
